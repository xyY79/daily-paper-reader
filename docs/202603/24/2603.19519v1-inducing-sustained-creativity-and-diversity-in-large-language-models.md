## 摘要
我们探讨了探索性搜索中一个尚未被广泛认知的子集，即用户为了寻找完美的婚纱、被忽视的研究课题或极具潜力的公司创意等，而展开的通常较长期的“搜索探索（search quest）”。当前大语言模型（LLM）的前几个输出可能有所帮助，但仅是一个开始，因为这种探索过程需要学习搜索空间，并在过程中评估许多多样且具有创造性的替代方案。尽管 LLM 编码了世界上极大部分的知识，但常见的解码方法仅针对具有正确答案的提示词进行了狭隘的优化，因此返回的结果大多是同质且常规的。其他方法（包括那些旨在增加少量答案多样性的方法）在搜索探索用户学到足以做出最终选择的知识之前，就开始出现重复，或者为每个提出类似问题的用户提供千篇一律的“创造力”。我们开发了一种新颖且易于实现的解码方案，可以在 LLM 中诱导持续的创造力和多样性，即使在无法访问 LLM 向量空间内部机制的情况下，也能根据需要生成任意数量的概念上独特的结果。该算法解锁了 LLM 广博的知识（包括正统和非正统的知识），远超模态解码路径。通过这种方法，搜索探索用户可以更快速地探索搜索空间并找到满意的答案。

## Abstract
We address a not-widely-recognized subset of exploratory search, where a user sets out on a typically long "search quest" for the perfect wedding dress, overlooked research topic, killer company idea, etc. The first few outputs of current large language models (LLMs) may be helpful but only as a start, since the quest requires learning the search space and evaluating many diverse and creative alternatives along the way. Although LLMs encode an impressive fraction of the world's knowledge, common decoding methods are narrowly optimized for prompts with correct answers and thus return mostly homogeneous and conventional results. Other approaches, including those designed to increase diversity across a small set of answers, start to repeat themselves long before search quest users learn enough to make final choices, or offer a uniform type of "creativity" to every user asking similar questions. We develop a novel, easy-to-implement decoding scheme that induces sustained creativity and diversity in LLMs, producing as many conceptually unique results as desired, even without access to the inner workings of an LLM's vector space. The algorithm unlocks an LLM's vast knowledge, both orthodox and heterodox, well beyond modal decoding paths. With this approach, search quest users can more quickly explore the search space and find satisfying answers.

### 2. 论文提出的方法论：Recoding-Decoding (RD)
*   **核心思想**：通过在解码循环中动态注入随机性，强制模型偏离常规的模态路径，从而访问其内部编码的广博但被掩盖的长尾知识。
*   **关键技术细节**：
    1.  **随机引导短语（Random Priming Phrases）**：在提示词开头随机添加类似“Related to [NOUN]”的短语（名词从常用词库中随机抽取），利用模型的“位置偏见”引导生成方向。
    2.  **随机转向 Token（Random Diverting Tokens）**：在每个新句子的开头注入随机的三个字母前缀（如“Pas”、“Tib”），迫使模型进入特定的语义子空间。
    3.  **模拟补全 API（Simulated Completion API）**：针对仅提供对话接口的模型，通过系统提示词模拟“续写”模式，以绕过对话模型对随机注入内容的“纠错”倾向。
    4.  **语法修正步骤**：由于注入了随机前缀，原始输出可能存在拼写错误，RD 增加了一个轻量级的语法修正步骤（Grammar Corrector）以保证文本流畅度。

### 6. 主要结论与发现
*   **性能飞跃**：RD 在保持高相关性（>0.95）的同时，生成的不重复概念数量远超所有基准方法。
*   **模型悖论**：**模型越先进（准确率越高），其标准解码的多样性反而越差**（概率分布更陡峭），但 RD 在更先进的模型上提升效果更显著，说明先进模型内部蕴含了更多知识，只是被常规解码路径掩盖了。
*   **持续性**：RD 能够支持长达 1,000 次运行而不陷入重复，而传统方法在几次运行后就会出现严重的模式坍塌。

### 7. 优点
*   **易于实现**：无需访问模型权重或向量空间，仅通过文本层面的干预即可实现。
*   **普适性强**：适用于各种 LLM，且能通过调整注入词库实现特定领域（如中国史、内容审核）的定制化引导。
*   **解决痛点**：直接针对 LLM 在创意工作中的“群体思维（Groupthink）”和“平庸化”问题。

### 8. 不足与局限
*   **Token 成本**：语法修正步骤增加了额外的 Token 消耗。
*   **相关性风险**：在某些极其狭窄的搜索空间（如“时尚配饰”）中，过度追求多样性可能导致生成结果的相关性略微下降。
*   **API 依赖**：在没有真实补全（Completion）API 的模型上，模拟效果虽好但仍略逊于原生补全接口。
*   **偏见风险**：虽然 RD 能挖掘非正统知识，但也可能在不受控的情况下挖掘出模型训练数据中的偏见或有害内容（尽管论文展示了其在内容审核中的潜力）。
