## 摘要
大语言模型（LLMs）在知识密集型任务中表现出幻觉。基于图的检索增强生成（RAG）已成为一种极具前景的解决方案，然而，现有方法在处理黑盒知识图谱（即模式和结构预先未知的图谱）时，面临着根本性的召回率和准确率限制。我们识别了导致召回率损失（语义实例化不确定性和结构路径不确定性）以及准确率损失（证据比较不确定性）的三个核心挑战。为了应对这些挑战，我们将检索任务形式化为最优信息子图检索（OISR）问题——这是群组斯坦纳树（Group Steiner Tree）的一个变体——并证明其为 NP-hard 和 APX-hard。我们提出了 BubbleRAG，这是一种无需训练的流水线，通过语义锚点分组、用于发现候选证据图（CEGs）的启发式气泡扩张、综合排序以及推理感知扩张，系统地优化了召回率和准确率。在多跳问答（QA）基准测试上的实验表明，BubbleRAG 取得了当前最先进的结果，在 F1 值和准确率上均优于强基准模型，同时保持了即插即用的特性。

## Abstract
Large Language Models (LLMs) exhibit hallucinations in knowledge-intensive tasks. Graph-based retrieval augmented generation (RAG) has emerged as a promising solution, yet existing approaches suffer from fundamental recall and precision limitations when operating over black-box knowledge graphs -- graphs whose schema and structure are unknown in advance. We identify three core challenges that cause recall loss (semantic instantiation uncertainty and structural path uncertainty) and precision loss (evidential comparison uncertainty). To address these challenges, we formalize the retrieval task as the Optimal Informative Subgraph Retrieval (OISR) problem -- a variant of Group Steiner Tree -- and prove it to be NP-hard and APX-hard. We propose BubbleRAG, a training-free pipeline that systematically optimizes for both recall and precision through semantic anchor grouping, heuristic bubble expansion to discover candidate evidence graphs (CEGs), composite ranking, and reasoning-aware expansion. Experiments on multi-hop QA benchmarks demonstrate that BubbleRAG achieves state-of-the-art results, outperforming strong baselines in both F1 and accuracy while remaining plug-and-play.

现有方法在黑盒场景下存在三大挑战：
*   **语义实例化不确定性**：查询概念在图中可能有多种异构表达，导致初始锚点定位困难（影响召回率）。
*   **结构路径不确定性**：缺乏模式指导，难以确定连接实体的最优关系链（影响召回率）。
*   **证据比较不确定性**：难以在多个候选子图中识别最具信息量的证据（影响精确度）。

### 2. 论文提出的方法论
论文将检索任务形式化为**最优信息子图检索（OISR）问题**（证明其为 NP-hard 和 APX-hard），并提出了 **BubbleRAG** 框架，包含以下五个关键步骤：
1.  **数据准备**：构建图谱时将关系文本直接嵌入边中，支持边级别的语义匹配。
2.  **语义锚点分组**：利用 LLM 提取显式关键词并推断隐式概念，通过“锚点专业化”和“模式松弛”技术，将查询映射为带权重的候选节点/边组。
3.  **候选证据图（CEG）发现**：提出**启发式气泡扩张（Bubble Expansion）算法**。这是一种多源、代价驱动的各向异性搜索，模拟气泡膨胀并在碰撞处融合，从而在复杂图中高效寻找连接多个锚点组的紧凑子图。
4.  **CEG 排序**：采用复合评分机制，结合**语义不和谐度**（节点与查询的平均距离）和**结构不完整性惩罚**（未覆盖锚点组的权重惩罚）对候选子图进行筛选。
5.  **推理感知扩展**：对排名靠前的 CEG 进行 LLM 引导的局部多步扩展，以捕获可能位于骨干路径之外的答案实体。

### 6. 论文的主要结论与发现
*   **SOTA 性能**：BubbleRAG 在所有基准测试中均取得领先，尤其在最复杂的 MuSiQue 数据集上，F1 值比强基准 HippoRAG2 高出约 8%。

*   **效率平衡**：虽然推理延迟高于 NaiveRAG 和 HippoRAG2（因为涉及多次 LLM 调用），但显著快于 ToG 等迭代搜索方法，且索引成本可控。

### 7. 优点
*   **理论支撑扎实**：将图检索问题提升到 OISR 理论高度，并给出了硬度证明。
*   **解决黑盒痛点**：不依赖预定义的 Schema，通过语义分组和启发式扩张有效解决了实体歧义和结构未知问题。
*   **即插即用**：无需微调模型，对现有图谱结构改动小，易于部署。
*   **灵活的查询语义**：通过调整惩罚因子 $\alpha$，可以轻松支持“与（AND）”、“或（OR）”及“比较”等多种复杂查询逻辑。

*   **推理延迟**：由于在锚点分组和推理扩展阶段多次调用 LLM，平均每条查询耗时约 21 秒，对于实时性要求极高的场景可能存在压力。
